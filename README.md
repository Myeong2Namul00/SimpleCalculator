# (C# 코딩) 나만의 계산기

## 개요
- C# 프로그래밍 학습
- 1줄 소개: 윈도우 기본 계산기의 기능을 구현한 애플리케이션
- 사용한 플랫폼:
  - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - TextBox: 계산 상황과 결과를 표시하는 컨트롤
  - Button: 계산기의 각각의 버튼 기능을 하는 컨트롤
  - Label: 계산기 제목을 표시하는 컨트롤
- 사용한 기술과 구현한 기능:
  - Visual Studio의 Designer를 활용하여 UI 구성
  - string 클래스를 활용하여 계산 상황과 결과를 문자열로 처리하는 기능 구현
  - button 컨트롤의 Click 이벤트를 활용하여 사용자 입력에 따른 계산 기능 구현
  - textbox 컨트롤의 Text 속성을 활용하여 계산 상황과 결과를 표시하는 기능 구현
  - switch 문을 활용하여 연산자에 따른 계산 기능 구현
  - 메소드로 코드를 구조화하여 유지보수성을 높이는 기능 구현
  - 각 컨트롤의 이름을 명확하게 지정하여 코드의 가독성을 높이는 기능 구현

# 각 과제별 실행 화면

## 실행 화면 (과제1)



- 과제 내용
	- TextBox(입력표시, 결과표시), Button(계산) 등을 적절히 배치합니다.
	- 숫자 Button 클릭 시 TextBox에 표시합니다. 2가지 방법으로 표시
	- 2개의 피연산자의 입력값을 Int로 바꾸어 더하기 계산을 수행하고, 그 결과를 저장합니다.
	- 계산 결과 값을 문자열로 반환하여 표시합니다.

- 구현 내용과 기능 설명
	- Designer를 활용하여 UI 구성
	- Button 컨트롤의 Click 이벤트를 활용하여 사용자 입력을 받는 기능 구현
	- TextBox 컨트롤을 활용하여 숫자를 표시하는 기능을 구현
	- 변수에 값을 저장하고 더하기 계산을 수행하는 기능 구현


## 실행 화면 (과제2)



- 과제 내용
	- 뺄셈(-), 곱셈(*), 나눗셈(/) 버튼 추가
	- 이벤트 연결
	- 각 버튼 클릭 시 연산자만 변경하여 동일 로직 적용

	- 각 사칙연산 버튼에 이벤트를 연결하여 사용자 입력을 받는 기능 구현
	- ApplyBinaryOperation 메소드를 활용하여 연산자에 따른 계산 기능 구현
	- 뺄셈, 곱셈, 나눗셈 버튼 클릭 시 연산자만 변경하여 동일 로직 적용 기능 구현
	- switch 문을 활용하여 연산자에 따른 계산 기능 구현


## 실행 화면 (과제3)



- 과제 내용
	- 계산기에 있는 수정/삭제 기능 구현
	- 다음의 사례로 설명 : 12 + 100 = 112
	- C버튼 기능 (현재의 모든 내용을 삭제하고 처음 상태로 되돌아감) 구현
	- CE버튼 기능 (마지막 입력한 피연산자 값을 삭제) 기능 구현
	- 100 입력 후에 Del 눌렀다면 100값이 통째로 삭제됨
	- Del버튼 기능 (마지막 입력된 글자 하나 값을 삭제) 기능 구현

- 구현 내용과 기능 설명
	- C버튼 클릭 시 TextBox의 내용을 초기화하여 처음 상태로 되돌아가는 기능 구현
	- CE버튼 클릭 시 마지막 입력한 피연산자 값을 삭제하는 기능 구현
	- Del버튼 클릭 시 마지막 입력된 글자 하나 값을 삭제하는 기능 구현
	- TextBox 컨트롤의 Text 속성을 활용하여 계산 상황과 결과를 표시
	- 문자열 조작을 통해 입력된 숫자와 연산자를 관리하는 기능 구현
